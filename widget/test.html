This page tests the Spikes feedback widget. Click the floating button in the bottom-right corner to leave feedback.

How to Use

Add the widget to any HTML page with a single script tag:

<script src="spikes.js"></script>

Or configure it with data attributes:

<script 
  src="spikes.js"
  data-project="my-project"
  data-position="bottom-right"
  data-color="#e74c3c"
></script>

Features

  • Page-level feedback with ratings (love/like/meh/no)
  • Element-level feedback via spike mode
  • Free-text comments
  • localStorage persistence
  • Works on file://, localhost, and https://
  • Configurable position and color
  • Less than 10KB gzipped
