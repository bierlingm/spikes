Why Spikes?

Zero friction

One script tag. No accounts. No build step. Works on file://, localhost, anywhere.

Precise feedback

Element-level spikes capture CSS selectors. Know exactly what reviewers are talking about.

Agent-friendly

CLI outputs JSON. Pipe to jq, feed to agents, automate your workflow.

Your infrastructure

Local-first by default. Deploy to your own Cloudflare for multi-reviewer sync.

Tiny footprint

Widget is 8KB gzipped. No dependencies. No tracking. No bloat.

Free forever

MIT licensed. All features included. Pay only if you want to support development.

Install the CLI

curl -fsSL https://spikes.sh/install.sh | sh
cargo install spikes
git clone https://github.com/bierlingm/spikes && cd spikes/cli && cargo install --path .

CLI in action

$ spikes inject ./mockups/
Injected widget into 10 files.
